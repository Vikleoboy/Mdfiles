# To People Who Are Still Doing Webdev

If you started something like I did, learning programming by yourself, you will likely end up doing webdev at some point. It is just unavoidable, especially in this age, and most people never escape this hell hole.
Just imagine a world where you have a JS library for everything, and 1000 other same copies of something similar doing the same exact thing. And for everything you do in programming, you have to learn something new, some new framework or library every day, some new EVERY FUCKING MORNING for the things we've been doing for ages, like making websites, making http servers, and god knows the list.
THIS IS NIGHTMARE FOR ME.
SUCH a world is full of rubbish and is rubbish itself! There is no certainty in this world, and the fact that there is always some abstractions on top of something else is just crazy to me!
And yes, I was limited to such world myself. This is what is wrong with languages like Python, JavaScript, they don't really introduce you to the long history of development, they never teach you how the computers really work. Just imagine ignoring all the work of so many people, actual code that made an impact in their time, that was actual innovation that they did, people whose code runs this world but is invisible to the eye of the ignorant.
